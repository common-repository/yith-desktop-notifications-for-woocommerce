== Description ==

YITH Desktop Notifications for WooCommerce gives you the possibility to receive real time notifications right on your PC screen letting you know about orders.

= Features: =

* Receive real time notifications about orders.

= PREMIUM FEATURES of YITH Desktop Notifications for WooCommerce: =

* Create five different types of notifications.
* Upload your custom notification icons.
* Upload your custom notification sounds.
* Allows notifying only those users with a specific role in your shop.

== Installation ==

1. Unzip the downloaded zip file.
2. Upload the plugin folder into the `wp-content/plugins/` directory of your WordPress site.
3. Activate `YITH Desktop Notifications for WooCommerce` from Plugins page.

YITH Desktop Notifications for WooCommerce will add a new submenu called "Desktop Notifications" under "YIT Plugins" menu. Here you are able to configure all the plugin settings.
